HTML

HTML (HyperText Markup Language) is the code that is used to structure a web page and its content. For example, content could be structured within a set of paragraphs, a list of bulleted points, or using images and data tables.

Document type declaration

!DOCTYPE html

instruction to the web browser about what version of HTML the page is written in

Difference Between HTML Tags and elements

HTML tags are used to hold the HTML element. HTML element holds the content.

Block level elements

A block-level element always starts on a new line, and the browsers automatically add some space before and after the element. A block-level element always takes up the full width available . Two commonly used block elements are p and div
